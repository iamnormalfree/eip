  The Learning Loop: How Improvements Flow

  "MAF Wednesday" - Weekly sync process:

  1. Collect improvements from all repos
     "We fixed this in nextnest"
     "We added that in roundtable"

  2. Review and categorize
     Core code improvement? Script fix? Documentation?

  3. Update the MAF repository
     Make the improvement
     Tag new version (v0.1.0 → v0.1.1)

  4. All repos can update
     git subtree pull --prefix=maf ...
     Everyone gets the improvement
